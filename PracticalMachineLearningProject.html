Executive Summary

In this report, analysis is done on HAR training dataset and a decision tree will be used to predict HAR testing dataset. The predictive model shall predict the manner in which they did the exercise. This is the “classe” variable in the training set. In this report, following key questions will be attended:

  • How the model is built.
  • How cross validation is used
  • The expected out of sample error
  • Justification on the choices made.

dim(training)

remove = c('X', 'user_name', 'raw_timestamp_part_1', 'raw_timestamp_part_2', 'cvtd_timestamp', 'new_window', 'num_window')
training.meaningful <- training.temp[, -which(names(training.temp) %in% remove)]

        write.table(x[i],file=filename,quote=FALSE,row.names=FALSE,col.names=FALSE)
    }
}
